# Todo App Backend

A secure Todo REST API built using Node.js, Express.js, SQLite, and JWT Authentication.

## Features
- User Registration
- User Login
- JWT Authentication
- Create Todo
- Read Todos
- Update Todo
- Delete Todo
- SQLite Database

## Tech Stack
- Node.js
- Express.js
- SQLite
- JWT
- bcryptjs

## Installation

git clone <repository-url>

cd chapter3

npm install

npm run dev

Server runs on:
http://localhost:7003
